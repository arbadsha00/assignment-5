What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document (usually HTML or XML) as a tree of objects, where each node in the tree corresponds to part of the document, such as elements, attributes, and text content.

How do you select an element from the DOM?

To select an element from the DOM, you can use several JavaScript methods. The getElementById('id') method selects an element by its id, while getElementsByClassName('class') selects all elements with a specific class name. Similarly, getElementsByTagName('tag') selects elements by their tag name. If you need more flexible selection, querySelector('selector') selects the first element that matches a CSS selector, and querySelectorAll('selector') selects all matching elements. These methods allow you to target and manipulate specific elements in the document.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the context of the DOM is a technique where a single event listener is added to a parent element rather than multiple event listeners to individual child elements. The event listener takes advantage of event bubbling (where an event propagates up through the DOM from the target element to its ancestors) to handle events for multiple child elements efficiently.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can use methods like getAttribute() to retrieve an attribute value or setAttribute() to modify it. For example, you might change the src attribute of an image using element.setAttribute('src', 'newImage.jpg') or remove an attribute with removeAttribute(). Alternatively, attributes like id and href can be modified directly as properties, such as element.id = 'newId'. For styling, you can adjust inline styles using the style property, for example, element.style.color = 'blue', or manage styles more efficiently by adding or removing CSS classes with classList.add() or classList.remove(). These techniques allow dynamic changes to the appearance and behavior of DOM elements.
